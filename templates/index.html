What's a chatbot?

A chatbot by definition is a computer program designed to simulate conversation with human users. Often over the internet. Currently, chatbots are are able to fulfil several roles. One example may be for an enterprise purpose such as customer support and administrative purposes. This could provide a cheap alternative to hiring staff to achieve the same purpose. Futhermore, chatbots are also used generally: for simple entertainment, conversations or maybe even an online friend/companion.

The Model

What is Neural Machine Translation and what is Attention?

The model consists of three main layers: An encoder, A transitional processing state and a decoder. The model first encodes the english based input into a vectorized, numerical vector which is then passed through several Long short term memory units [LSTMs] which are effectively more efficient Recurrent Neural networks that don't suffer from the vanishing gradient problem. The output is then passed through the decoder which then decodes the network's prediction back into english.

Attention consists of creating a probability distribution between the LSTMs current memory and condensing inputs to form ideas. This prevents the model from just forming a one to one correlation (linear) between words and rather uses the context of several words to form an output
